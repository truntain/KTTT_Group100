# Nhóm 100 - Nghiên cứu và Ứng dụng Thuật toán Tối ưu

Chào mừng đến với kho lưu trữ mã nguồn của Nhóm 100. Dự án này tập trung vào nghiên cứu thuật toán **Grey Wolf Optimizer (GWO)**, phát triển các biến thể lai ghép và ứng dụng vào bài toán thực tế trong mạng 6G.

## Danh sách các Module

### 1. [GWO-GA Hybrid Core (Legacy)](./GWO-GA_Code)
* **Mô tả:** Triển khai thuật toán lai ghép giữa GWO và Giải thuật di truyền (Genetic Algorithm).
* **Ngôn ngữ:** C (Core) và Python (Visualization).
* **Mục đích:** Nghiên cứu cơ bản và so sánh hiệu năng hội tụ.

### 2. [JCAS Multibeam Optimization](./jcas_GWO)
* **Mô tả:** Ứng dụng GWO để giải quyết bài toán tối ưu hóa đa chùm tia cho hệ thống JCAS (Joint Communication and Sensing).
* **Ngôn ngữ:** Python.
* **Mục đích:** Ứng dụng thực tiễn, tối ưu hóa SINR và độ lợi Radar đồng thời.
